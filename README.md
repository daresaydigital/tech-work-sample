
## Getting Started

To run this app, you are going to need to get [Google Maps JavaScript API](https://developers.google.com/maps/documentation/javascript/)

## Setup

```
$ npm install
```

## Configs

You need to set WETHER_API_KEY and GOOGLE_API_KEY in config
```
$ cd src/config
$ gedit index.js
```


## Running

```
$ npm start
```
## Test

```
$ npm run test
```

## Build

```
$ npm run build
```
